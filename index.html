    FAQs
    We are here to help you
    Can I use Waze in split screen?
    Waze officially don't support any API for third party app like vivid so we cannot show waze map view on the left but when you use Waze Waze will automatically replace Google map as default map and show as the second app on bottom domain bar. And any search or drive shortcut wi...
    Why I cannot see Radio and BT page on my device?
    Radio and Bluetooth functions are bond to hardware type. If you are using headunits vivid supported like PX6 PX5 etc you'll get Radio and Bluetooth music and phone call pages and functions works very well. If you are using other type of devices those function will..
    My media player is not in the media source list?
    Some media app don't follow Android media interface so we cannot control them by default let us know your media app which is not supported to try to find solution.
    Why split screen Google map don't have active route when in navigation?
    Vivid's using Google map sdk for split map view while unfortunately Google don't provide API to sync the active route information with Google maps app.
    Change bottom domain bar apps and order?
    You can long press bottom domain app to unpin or drag to order. You can also pin any app you want from all apps page.
    Any place for feature request or bug report?
    You can feedback in vivid launcher settings->Feedback or email us or feedback in Xda forum which lots of peers already talk about vivid for long time.
